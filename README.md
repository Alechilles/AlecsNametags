# Alec's Nametags!

A lightweight mod that adds a craftable nametag item for naming companions using **Alec's Tamework**.

## Features
- Adds **Nametag**, a craftable naming tool.
- Default naming rules:
  - requires the NPC to be tamed
  - requires ownership, but also allows unowned tamed NPCs
  - allows renaming and replaces existing names
  - supports letters, numbers, and spaces

## Usage
1. Craft at a **Farmer's Workbench**
  - 2x **Light Leather**
  - 1x **Gold Bar**
2. Hold **Nametag**.
3. Left click on a valid NPC to name it.
4. Enter the desired name in the naming prompt.

## Requirements
- Alechilles:Alec's Tamework! 2.4.x
